# RMMV
RPG Maker MV Plugins (personal work)

# これは何？
SakuraCrowd が個人で作成した RPG Maker MV (RPG ツクール MV) 用のプラグイン集です。  
サブフォルダ単位でプラグインを公開します。  

# プラグインの説明
## SCTitleText
タイトルシーンにパラメータで指定したフォントと文字列でテキストを表示します。  

## SCGetDistance
イベントとイベント、イベントと座標との距離を計算します。  
プラグインコマンドで呼び出して使います。
